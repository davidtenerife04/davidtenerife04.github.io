Básicos Redes Archivos Sistema

Comandos Básicos

  • dir — Listar archivos y carpetas.
  • cd — Cambiar de directorio.
  • cls — Limpiar la pantalla.
  • exit — Salir del terminal.

Comandos de Redes

  • ipconfig — Mostrar configuración IP.
  • ping [dirección] — Comprobar conexión.
  • tracert [dirección] — Rastrear ruta de paquetes.
  • netstat — Ver conexiones de red activas.

Comandos de Archivos y Carpetas

  • mkdir [nombre] — Crear nueva carpeta.
  • del [archivo] — Eliminar un archivo.
  • rmdir [carpeta] — Borrar carpeta vacía.
  • copy [origen] [destino] — Copiar archivos.

Comandos del Sistema

  • tasklist — Ver procesos activos.
  • taskkill /IM [nombre].exe — Terminar procesos.
  • shutdown /s /t 0 — Apagar el equipo.
  • systeminfo — Mostrar información del sistema.

    rmdir /s /q "C:\ruta\a\la\carpeta" o archivo Eliminar directorio o archivo
